About the project

We are all web development students at DCI (Digital Career Institute) in Berlin. This is our first group project where we try to get our already learned topics and stuff into our first product.

The Movie Library is an IMDb clone with a full Internet Movie Database which provides information about millions of films and television programs as well as their cast and crew. Our group decided to experiment working on IMDb website using IMDb APIs and giving a nice shape with helpful features.

Features

Searching a movie and sorting of films
Description of the provided films with rating
My favorite list with own rating
